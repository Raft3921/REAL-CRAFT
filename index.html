遊び方 / 実装内容

1. タイトル統一
タイトルはすべて REAL CRAFT に統一。タイトル画面も差し替え済み。
2. レンの両手とHUD
一人称時は両手と装備を表示。ライフ / マナ / レベル / スタミナ丸ゲージを搭載。
3. ブロック系
ブロック、鉱石、木材、ガラス、古代ブロック、農地を配置可能。
4. ツール・武器・設備
ツルハシ・斧・剣・杖、作業台・かまど・料理台をホットバー登録可能。
5. モブ
動物モブと敵モブを出現。周囲を歩き回ります。
6. 属性・魔法・熟練度
火 / 水 / 風 / 土 / 光 / 闇の属性熟練度を表示。杖装備中は魔力ゲージを確認可能。
7. 農業・料理・遺跡
畑、料理台、古代遺跡風の建造物をワールドに生成。
8. 視点切替
Vキー または画面ボタンで一人称 / 三人称を切替。
