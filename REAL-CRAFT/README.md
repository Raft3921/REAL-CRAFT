# REAL CRAFT α 0.3

α 0.2をさらに強化した、戦闘・ドロップ・クラフト対応版です。

## 今回追加した機能

- 敵・動物モブへの攻撃
- モブごとの体力
- モブ撃破とドロップ
- 木材・石・鉄・魔結晶・食料の素材管理
- ブロック破壊で素材獲得
- クラフト画面
- クラフトレシピ
- 作業台・かまど・料理台・武器・魔法杖の作成
- クラフト状態と素材数のセーブ
- iPad用「攻」「作」ボタン
- PC用 F攻撃 / Cクラフト

## 操作

### iPad
- 左パッド：移動
- 右側ドラッグ：視点
- 跳：ジャンプ
- 壊：破壊
- 置：設置
- 攻：攻撃
- 作：クラフト
- 視：視点切替
- ≡：ポーズ

### PC
- WASD：移動
- Shift：ダッシュ
- Space：ジャンプ
- 左クリック：破壊
- 右クリック：設置
- F：攻撃
- C：クラフト
- B：インベントリ
- V：一人称 / 三人称
- 1〜8：ホットバー
- Esc：ポーズ
