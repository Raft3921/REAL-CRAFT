# REAL CRAFT α 0.4

## 今回の修正

- 更新が分かるようにタイトル画面へ **BUILD 0.4 — 操作反転修正版** を表示
- ブラウザキャッシュ抑制用の設定を追加
- 視点操作と移動操作の反転設定を追加
- タイトル画面の「設定」から、左右・上下・前後を個別に切替可能
- ZIP内を直下構成に変更し、古い `index.html` を確実に上書きしやすくした

## 重要

このZIPは、展開すると直下に以下が出ます。

- `index.html`
- `README.md`
- `.nojekyll`
- `assets/`

この4つを、GitHubリポジトリの一番上にある古いファイルへ**上書き**してください。

GitHub Pagesで変化しない場合は、Safariでページを開いたまま更新ボタンを長押しし、
「デスクトップ用Webサイトを表示」またはタブを閉じて開き直してください。

タイトル画面に `BUILD 0.4 — 操作反転修正版` が表示されれば更新成功です。
