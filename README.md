# REAL CRAFT 1.1 PC専用・3Dモデル強化版

## 改善内容
- タイトル画像を新しい画像へ変更
- 画像全体が切れないように contain 表示
- タイトルの矢印をマウスカーソルの選択位置へ追従
- ボタンへ約0.28秒カーソルを置くだけで選択
- クリック操作も使用可能
- 最新セーブデータ欄を実際のセーブ内容で更新
- 完全PC版としてモバイル操作UIを非表示
- 移動をW/S/D/Aへ固定
- ゲーム画面から属性ショートカットを削除
- HUDをライフ・レベル・経験値・丸型スタミナゲージ中心へ整理
- 三人称用のレンのプロシージャル3Dモデルを追加
- 頭、髪、胴体、腕、脚、短パンを立体化

## 操作
- W：前
- S：後ろ
- D：右
- A：左
- Shift：走る
- Space：ジャンプ
- マウス：視点
- 左クリック長押し：採掘
- 右クリック：設置
- E：使用
- F：攻撃
- V：視点切替
