# REAL CRAFT α 0.6

今回は見た目を中心にリアル寄りへ強化しました。

## 強化内容
- ACESトーンマッピング
- 柔らかい高解像度シャドウ
- 時間で動く太陽
- 雲の移動
- 水面テクスチャの移動
- 草・石・木・ガラスの簡易プロシージャルテクスチャ
- 木、岩、草むらの環境オブジェクト
- 森林・雪原・砂漠・沼地ごとの装飾密度変更
- 一人称の腕を角張った箱型から丸みのある形へ変更
- 歩行時の手の揺れ
- 金属・ガラス・光源の質感調整

## 正直な範囲
ブラウザでiPadでも動くようにしているため、家庭用ゲーム機並みの高密度モデルや
超高解像度テクスチャではありません。今回は動作を保ちながら、
光、影、素材感、地形の密度を改善したリアル寄りのα版です。
